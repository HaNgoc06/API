# Thông tin sinh viên
- **Họ và tên:** Nguyễn Thị Ngọc Hà
- **MSSV:** 24120299
- **Lớp:** 24CTT5

# Giới thiệu mô hình
- **Model:** [EnViT5-Translation](https://huggingface.co/VietAI/envit5-translation) của VietAI.
- **Chức năng:** Dịch thuật song ngữ Anh - Việt sử dụng kiến trúc T5.

# Hướng dẫn cài đặt
1. Clone dự án: `git clone <link-repo>`
2. Cài đặt thư viện: `pip install -r requirements.txt`

# Hướng dẫn chạy
1. Chạy API Server: `python main.py`
2. Chạy file kiểm thử: `python test_api.py` (Lưu ý thay URL Pinggy tương ứng).
